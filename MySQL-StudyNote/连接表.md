#### JOIN

join用于加入其它表

```mysql
select order_id,first_name,last_name
from orders
join customers on orders.customer_id=customers.customer_id
```

-- 多个表中的同一类 需要对其限定

```mysql
select order_id,first_name,last_name,customers.customer_id -- 不然mysql不知道你要从俩表里的哪个表里取id列
from orders
join customers on orders.customer_id=customers.customer_id
ORDER BY order_id
```

-- 可以给表起别名进行书写上的简化

```mssql
select order_id,first_name,last_name,c.customer_id
from orders o
join customers c
	on o.customer_id=c.customer_id
ORDER BY order_id
```

-- 跨数据库加入：只需为不属于当前数据库的表添加前缀

```mysql
-- 加入其它数据库的表

	 -- 当你使用 sql_store 库时
use sql_store;
select * 
from order_items oi
join sql_inventory.products sp
	 on oi.product_id = sp.product_id
	 
	 -- 当你使用 sql_inventory 库时
use sql_inventory;
select * 
from sql_store.order_items oi
join products sp
	 on oi.product_id = sp.product_id
```

-- join表自己本身

```mysql
use sql_hr;
select 
	e.employee_id,
    e.first_name,
    m.first_name as manager
from employees e
join employees m
	 on  m.employee_id = e.reports_to 
```
