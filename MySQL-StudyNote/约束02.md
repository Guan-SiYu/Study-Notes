## 唯一约束

约束修饰字段的值不可以重复

```
create table User1(
	id int,
  name varchar(20)
);
alter table user1 add unique(name);
+-------+-------------+------+-----+---------+-------+
| Field | Type        | Null | Key | Default | Extra |
+-------+-------------+------+-----+---------+-------+
| id    | int         | YES  |     | NULL    |       |
| name  | varchar(20) | YES  | UNI | NULL    |       |
+-------+-------------+------+-----+---------+-------+
```
